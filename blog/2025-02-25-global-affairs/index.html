Capillaries: dissecting $66b allocated by Global Affairs Canada for 1996-2047

International assistance "projects" funded by Global Affairs Canada

Global Affairs Canada website gives access to some information about "projects" it funds with taxpayers' money. As of February 2025, the site offers information about 7769 "projects", including:

  • start and end date
  • total amount allocated
  • distribution by country/region
  • distribution by sector
The funds allocated for each project are funneled through a "partner" organization. As of February 2025, the website offers information about 1258 "partners". Some "partners" participate in the allocation of funds for multiple "projects".

An integration test "global_affairs" was recently added to Capillaries. The test denormalizes the data obtained from the website and prepares it for further investigation with data analysis tools like Tableau. All data files can be downloaded here (6mb).

What does it do, exactly?

For each "project", global_affairs integration test transforms JSON data holding country/region funds distribution and sector funds distribution into denormalized table that can be used for data analysis. For example, to analyze funds allocation data by country, the following funds distribution JSON:

project_id partner_id amount start_date end_date country_amt_json
